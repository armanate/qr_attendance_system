# سیستم حضور و غیاب با کد QR

یک سیستم حضور و غیاب مبتنی بر Django که از کدهای QR برای پیگیری حضور دانش‌آموزان استفاده می‌کند. این سیستم دارای رابط کاربری وب به زبان فارسی با جهت راست به چپ (RTL) می‌باشد.

## ویژگی‌ها

- مدیریت دانش‌آموزان (افزودن، مشاهده دانش‌آموزان)
- تولید کد QR برای هر دانش‌آموز
- اسکن کد QR برای ثبت حضور
- نمایش دانش‌آموزان حاضر و غایب
- پایگاه داده SQLite برای ذخیره‌سازی اطلاعات
- رابط کاربری وب به زبان فارسی با پشتیبانی از RTL

## نیازمندی‌ها

- پایتون 3.6 یا بالاتر
- Django 3.2 یا بالاتر
- وابستگی‌های ذکر شده در `requirements.txt`

## نصب

1. این مخزن را کلون کنید یا فایل‌ها را دانلود کنید
2. وابستگی‌های مورد نیاز را نصب کنید:

```bash
pip install -r requirements.txt
```

## استفاده

اجرای برنامه:

```bash
python manage.py runserver
```

سپس به آدرس `http://127.0.0.1:8000/` در مرورگر خود مراجعه کنید.

### مدیریت دانش‌آموزان

1. به صفحه "دانش‌آموزان" بروید
2. برای افزودن دانش‌آموز جدید، روی دکمه "افزودن دانش‌آموز جدید" کلیک کنید
3. نام و شناسه دانش‌آموز را وارد کنید و روی "افزودن دانش‌آموز" کلیک کنید
4. برای ایجاد کد QR، روی دکمه "تولید کد QR" در کنار نام دانش‌آموز کلیک کنید
5. برای حذف دانش‌آموز، روی دکمه "حذف" در کنار نام دانش‌آموز کلیک کنید و تأیید کنید

### ثبت حضور و غیاب

1. به صفحه "اسکن کد QR" بروید
2. برای فعال کردن دوربین روی "شروع اسکن" کلیک کنید
3. کدهای QR دانش‌آموزان را جلوی دوربین قرار دهید تا حضور آنها ثبت شود
4. پس از اتمام، روی "توقف اسکن" کلیک کنید

### مشاهده حضور و غیاب

1. به صفحه "حضور و غیاب" بروید
2. لیست دانش‌آموزان حاضر و غایب را مشاهده کنید
3. برای بازنشانی لیست حضور و غیاب روی "بازنشانی حضور و غیاب" کلیک کنید
4. برای چاپ لیست حضور و غیاب روی "چاپ لیست" کلیک کنید

## ساختار پروژه

- `attendance/`: اپلیکیشن اصلی Django برای سیستم حضور و غیاب
  - `models.py`: تعریف مدل‌های داده
  - `views.py`: توابع نمایش و پردازش درخواست‌ها
  - `forms.py`: فرم‌های ورود داده
  - `urls.py`: تعریف URL‌های اپلیکیشن
- `attendance_project/`: تنظیمات پروژه Django
- `media/qrcodes/`: دایرکتوری برای ذخیره کدهای QR تولید شده
- `templates/`: قالب‌های HTML برای رابط کاربری وب
  - `base.html`: قالب پایه با پشتیبانی RTL
  - `attendance/`: قالب‌های مختص اپلیکیشن حضور و غیاب
- `static/`: فایل‌های استاتیک (CSS، JavaScript)
- `manage.py`: اسکریپت مدیریت Django
